What it does

Being altruistic is a virtue, but can it be made into a game ? With this thought, we were inspired to build a mobile app to "gamify" altruism itself !

Our app, called Give Thanks, shows the user their closest charities, donate and compare their altruistic history with their friends !

How we built it

From the beginning to the end, we did it all over 24 hours ! We started off our project by painstakingly scraping a charity website which hosted a vast amount of data about the local charities in and around Chicago, IL. This can be expanded to a statewide or even countrywide in a similar manner.

We then began analysing all of this data, finding patterns that could be interpreted and explained. With this, we had a good idea of what we could stream line our backend database - MongoDB, driven by Node.js with Express.js to have maximum speed and efficiency for the users. Hosted on Heroku, this database streamlines access to large amounts of data and makes the entire framework scalable.

Backend matters, but users like awesome apps even more ! So we then set about designing a custom iOS application from the ground up. Designed for iOS 11.2, using Swift 4, we designed an efficient mobile app that will invite the user to use it constantly !

Challenges we faced

Right from the get go, we had an uphill battle. Scraping a website as noisy as that was a very challenging task, requiring constant updates and fixes to clean the incoming data.

After this, we had to integrate our dataset with our MongoDB database. Interfacing our database with the Heroku cloud platform-as-a-service was a major challenge, due to vague error messages, extremely limited tolerance to faulty queries etc.

Designing the iOS app was another major task. Having divided our workforce evenly to work on the backend and the app, the two developers tasked with designing the app had to face the daunting task of merging various external UI elements from various open source projects and interfacing with the still-nascent backend.

Accomplishments that we're proud of

What we Learned

The intricate task of building an entire functional project from the ground up was an eye-opening experience for all of us. Never before had we been tasked with such a vast variety of tasks we were unfamiliar with such as scraping websites, designing front end and backends simultaneously and even coming up with the core idea, all within such a restrictive time-frame.

What's next for GiveThanks

Next, we want to remove the kinks from the project - namely the large burdon on the backend to be reduced to make it more friendly to network-strapped users and to streamline the database even further to improve the load times of the application.

Built with

We used several technologies to complete this project. The dataset was scraped using the python scraping library - "Scrapy". Using Scikit-learn, we were able to cluster and understand some redundancies in the scraped data and remove them before designing our database.

We used a NoSQL database - MongoDB - backed by Node.js and Express.js to quickly build our backend. Simultaneously, we used XCode to quickly begin designing and implementing the logic to interface the the backend, and a veritable source of external open source UI elements.
